 🚀 AI Video Generator Landing Page

A modern, responsive AI SaaS landing page built using **HTML + Tailwind CSS**, fully generated with AI.

📌 Overview

This project is a high-quality landing page designed for AI video generator tools. It includes all essential sections required for a SaaS product and focuses on **conversion, design, and performance**.

✨ Features

* Responsive design (Mobile + Desktop)
* Modern dark UI
* Smooth animations
* Glassmorphism cards
* Pricing section
* Testimonials & FAQ
* CTA-focused layout

 🛠 Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
